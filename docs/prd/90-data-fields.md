# 全局字段设计

> 权威来源：`招聘管理系统_功能详细清单_v3.0.docx`。本分册是所有模块开发的数据/权限横切约束。

## 附录A：全局字段设计

### A.1 简历结构化字段

简历导入后自动提取并结构化存储以下字段：

<!-- table:80 rows:17 cols:2 -->
| 字段 | 说明 |
| --- | --- |
| 姓名 | 从简历中提取的候选人姓名 |
| 身份证号 | 从简历中提取的身份证号码（加密存储，用于去重） |
| 手机号 | 从简历中提取的手机号码（脱敏显示） |
| 邮箱 | 从简历中提取的电子邮箱 |
| 年龄 | 从简历中提取的年龄 |
| 性别 | 从简历中提取的性别（男/女） |
| 最高学历 | 从简历中提取的最高学历（高中/大专/本科/硕士/博士） |
| 院校规格 | 从简历中提取的院校规格（985/211/双一流/本科/大专） |
| 工作年限 | 从简历中提取的工作年限 |
| 期望职位 | 从简历中提取的期望岗位 |
| 期望行业 | 从简历中提取的期望行业 |
| 期望最低薪资 | 从简历中提取的期望最低月薪 |
| 期望最高薪资 | 从简历中提取的期望最高月薪 |
| 现居地 | 从简历中提取的当前居住城市 |
| 求职状态 | 从简历中提取的求职状态 |
| 到岗时间 | 从简历中提取的到岗时间 |

### A.2 客户公司字段

<!-- table:81 rows:9 cols:2 -->
| 字段 | 说明 |
| --- | --- |
| 公司名称 | 客户公司全称（必填） |
| 联系人 | 主要对接人姓名（必填） |
| 联系电话 | 联系电话（必填） |
| 联系邮箱 | 联系邮箱 |
| 公司地址 | 公司注册地址 |
| 合作周期 | 合作起止时间 |
| 状态 | 招聘中/空闲/失效 |
| 备注 | 备注信息 |

### A.3 项目字段

<!-- table:82 rows:9 cols:2 -->
| 字段 | 说明 |
| --- | --- |
| 项目名称 | 项目/岗位组名称（必填） |
| 所属客户公司 | 项目所属客户公司（必填） |
| 项目状态 | 招聘中/招聘完毕/招聘中止 |
| 项目等级 | A级、B级、C级 |
| 招聘人数 | 计划招聘人数 |
| 工作地址 | 工作地点 |
| 岗位需求描述 | 岗位职责和任职要求描述 |
| 备注 | 备注信息 |

### A.4 岗位字段

<!-- table:83 rows:10 cols:2 -->
| 字段 | 说明 |
| --- | --- |
| 岗位名称 | 招聘职位名称（必填） |
| 所属项目 | 岗位所属项目（必填） |
| 紧急程度 | 高（红色）、中（黄色）、低（绿色） |
| 招聘人数 | 该岗位计划招聘人数 |
| 薪资范围 | 最低薪资-最高薪资 |
| 工作地点 | 工作地点 |
| 年龄要求 | 从求职者标签字典中选择 |
| 学历要求 | 从标签字典中选择 |
| 工作经验 | 从标签字典中选择 |
