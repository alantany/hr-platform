# 模块九：统计与分析

> 本分册是可独立交给开发 agent 的模块 PRD。权威来源为 `招聘管理系统_功能详细清单_v3.0.docx`；若与旧原型理解或现有页面实现冲突，以产品最终 DOCX 为准。

## Agent 任务包

**任务目标**：按权限范围提供核心指标、趋势图、转化漏斗、排行榜、维度统计和报表导出。

**本模块负责**：

- 数据大屏核心指标、趋势、漏斗
- 员工/岗位/客户/组长排行
- 客户、组长、求职者维度统计
- 统计报表导出

**本模块不负责**：

- 基础业务数据录入
- 权限规则配置
- AI 算法实现

**必须先读取的横切文档**：

- `PRD.md`
- `docs/prd/90-data-fields.md`
- `docs/prd/91-permission-matrix.md`

**建议并行开发拆分**：

- 前端 agent：页面、表单、列表、筛选、状态展示、空态/失败态。
- 后端 agent：API、权限过滤、状态流转、操作日志、导入/导出或异步任务。
- 数据库 agent：模型、索引、枚举、关联表、迁移和种子数据。
- 测试 agent：接口级验收、权限矩阵、状态流转、边界条件和失败场景。

## 模块定位

统计管理模块为系统提供全方位的数据统计分析能力，通过数据大屏、排行榜、多维度统计和报表导出等方式，帮助管理者全面了解招聘运营情况。

## 独立边界

### 上游依赖

- 权限管理体系
- 求职者数据池
- 项目管理
- 推荐与交付管理
- 评价体系
- 质保期管理

### 本模块核心数据对象

- 统计指标
- 趋势数据
- 漏斗数据
- 排行榜
- 报表导出任务

### 对外输出

- 面向其他模块提供经过权限过滤的数据查询结果。
- 对关键业务动作输出可追溯的状态变化和操作日志。
- 对通知、统计、AI 或导出模块输出明确的业务事件或结果数据。

## 功能拆解

### 1. 数据大屏-核心指标

- 功能描述：完成业务数据的操作管理。进入统计管理 → 查看数据大屏 → 显示6项核心指标卡片
- 菜单/入口：统计管理 → 数据大屏
- 界面元素：无（系统自动加载）
- 输出结果：核心指标卡片展示
- 权限说明：所有用户按数据权限范围查看

**操作流程**

1. 进入统计管理
2. 查看数据大屏
3. 显示6项核心指标卡片

**业务规则**

- 管理员：简历总量、今日新增、活跃简历、待处理任务、本月推荐、本月录用；组长：授权范围简历总量、授权范围今日新增、团队推荐总量、团队转化率、客户公司数、项目数；操作员：个人简历总量、今日新增、今日面试、个人推荐量、个人转化率、待处理任务；卡片旁显示趋势箭头和百分比变化

## 推荐接口与后端边界

- 列表查询接口必须支持分页、排序、筛选、关键词搜索和数据权限过滤。
- 详情接口必须校验当前用户是否可访问该业务对象。
- 新增/编辑/删除/状态流转接口必须记录操作日志。
- 批量、导入、导出、AI、通知等长任务建议建任务表或事件表，保留状态、错误信息和重试线索。
- 所有枚举值优先从产品规格和标签字典中抽取，不允许前后端各自硬编码不同口径。

##### 1、业务需求

数据大屏-核心指标是统计管理模块为系统提供全方位的数据统计分析能力，通过数据大屏、排行榜、多维度统计和报表导出等方式，帮助管理者全面了解招聘运营情况。中的重要功能组件。该功能主要解决以下业务问题：

在人力公司招聘管理业务场景中，统计分析需要此功能来提供数据可视化和决策支撑。通过数据大屏功能，系统实现了业务效率的提升和数据管理的规范化，显著提升了业务操作效率和数据管理的规范性。

数据大屏功能的设计充分考虑了不同角色用户（超级管理员、组长、操作员）的实际操作需求，确保功能定位准确、业务逻辑清晰、操作便捷高效。同时，该功能与其他模块（如各业务模块的数据汇总与分析）形成紧密的业务关联，共同构成完整的招聘管理闭环。

##### 2、功能说明

数据大屏-核心指标功能定位于招聘管理的核心功能，是数据大屏体系中的核心功能组件。

统计报表导出是统计管理模块为系统提供全方位的数据统计分析能力，通过数据大屏、排行榜、多维度统计和报表导出等方式，帮助管理者全面了解招聘运营情况。中的重要功能组件。该功能主要解决以下业务问题：

在人力公司招聘管理业务场景中，统计分析需要此功能来提供数据可视化和决策支撑。通过统计报表导出功能，系统实现了业务效率的提升和数据管理的规范化，显著提升了业务操作效率和数据管理的规范性。

统计报表导出功能的设计充分考虑了不同角色用户（超级管理员、组长、操作员）的实际操作需求，确保功能定位准确、业务逻辑清晰、操作便捷高效。同时，该功能与其他模块（如各业务模块的数据汇总与分析）形成紧密的业务关联，共同构成完整的招聘管理闭环。

##### 2、功能说明

统计报表导出功能定位于招聘管理的核心功能，是统计报表导出体系中的核心功能组件。

该功能提供以下核心能力：
1. 支持业务数据的完整管理
2. 支持业务数据的查询和操作
3. 操作结果实时生效

统计报表导出功能与其他功能的关联关系：该功能与与其他业务模块联动形成完整业务流程紧密关联，共同支撑招聘管理的核心业务流程的完整业务流程。通过统计报表导出功能，操作员可以完成业务数据的操作管理，为后续的后续的推荐、交付和管理工作奠定数据基础。

功能权限方面，该功能有明确的角色权限限制。所有操作均记录操作日志，确保操作的可追溯性。

1. 在统计管理页面
